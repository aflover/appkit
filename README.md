# veak

Veak is desined as app components for vue and electron and is still in development 

### Using npm
```
npm install veak
```

### Live Demo

[demo](https://aflover.github.io/veak/demo)

### Implements Components
``` js
let components = {
	Btn,
	CheckBox,
	Dropdown,
	DropMenu,
	DropSelect,
	Flayout,
	FlexWindow,
	GridCol,
	GridRow,
	Icon,
	MenuIter,
	NavGroup,
	PureInput,
	PureSelect,
	RadioGroup,
	SimpleTab,
	SimpleTable,
	FormControl,
}
```
