# veak

app components for vue and electron

### Using npm
```
npm install veak
```

### Live Demo

[demo](./demo)

``` js
	Dropdown,

	Flayout,

	GridCol,
	GridRow,
	Icon,
	MenuIter,
	NavGroup,
	PureInput,
	PureSelect,
	RadioGroup,
	SimpleTab,
	SimpleTable,
	FormControl,
}
```
